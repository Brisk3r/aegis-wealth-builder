Deploy Aegis Developer Hub inside your private cloud or corporate firewall with SSO (Okta/SAML) integration, dedicated Docker containers, and team workspace sharing.

Docker & Docker Compose Deployment

1. Quick Start with Docker Compose:
git clone https://github.com/Brisk3r/aegis-wealth-builder.git aegis-hub
cd aegis-hub
docker compose up -d
2. Build & Run Standalone Container:
docker build -t aegis-hub:latest .
docker run -d --name aegis-hub -p 80:80 aegis-hub:latest
3. Security & Air-Gapped Verification:
Aegis Hub runs 100% inside client browser memory. No telemetry or analytics data leaves your corporate network when deployed behind an Nginx or Traefik reverse proxy.
