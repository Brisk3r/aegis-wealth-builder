No. 100% of calculation, PDF rendering, image compression, and RegEx processing occurs locally inside your browser session. Zero bytes leave your machine.

Can I use generated Tailwind UI cards, SVG vectors, and code snippets in commercial applications?

How does local browser session history work for the Aegis Task Tracker?

The Task Tracker leverages HTML5 LocalStorage and IndexedDB within your browser session, with optional JSON import/export backup features.
